<h2>🇧🇷 Português</h2>

<h1>Educap 📚</h1>

<p>Um aplicativo mobile desenvolvido para facilitar a comunicação e organização entre alunos, professores e a direção de uma escola.</p>

<hr>

<h2>Sobre o projeto</h2>

<p>O Educap surgiu da ideia de que um sistema escolar seria muito mais prático no celular do que em um site, já que o aluno que quer ver a tarefa de amanhã ou o professor que quer criar uma tarefa vai pegar o celular, não o notebook.</p>

<p>O app é dividido em três perfis de acesso: <b>Aluno</b>, <b>Professor</b> e <b>Diretor</b>, cada um com funcionalidades específicas. Algumas imagens do aplicativo estão disponíveis na pasta <code>assets</code> na raiz do projeto.</p>

<hr>

<h2>Funcionalidades</h2>

<h3>🎒 Aluno</h3>
<ul>
  <li>Visualização das aulas semanais em um grid com indicador de horário atual</li>
  <li>Acesso às tarefas da turma com filtros e busca</li>
  <li>Dashboard diário com próxima aula e resumo de tarefas</li>
  <li>Feed de publicações da escola</li>
  <li>Visualização da turma com colegas e professores cadastrados</li>
</ul>

<h3>👨‍🏫 Professor</h3>
<ul>
  <li>Criação, edição e remoção de tarefas por turma</li>
  <li>Visualização das aulas semanais com identificação de turma</li>
  <li>Acesso aos detalhes de cada turma</li>
  <li>Dashboard diário e feed de publicações da escola</li>
</ul>

<h3>🏫 Diretor</h3>
<ul>
  <li>Controle de acesso: geração de chaves para cadastro e verificação de novos usuários</li>
  <li>Gerenciamento de turmas</li>
  <li>Montagem do grid de aulas semanais por turma</li>
  <li>Criação de publicações para comunicação com alunos e professores</li>
</ul>

<hr>

<h2>Tecnologias utilizadas</h2>

<b>Backend</b>
<ul>
  <li>Java com Spring Boot</li>
  <li>Spring Security + JWT</li>
  <li>PostgreSQL + PgAdmin4</li>
  <li>Docker</li>
  <li>Swagger OpenAPI</li>
</ul>

<b>Frontend</b>
<ul>
  <li>React Native</li>
</ul>

<hr>

<h2>Como rodar</h2>

<h3>Pré-requisitos</h3>
<ul>
  <li>Docker instalado</li>
  <li>Android Studio instalado</li>
</ul>

<h3>Backend</h3>

<p>Abra a pasta do backend (<code>EducapBack/Educap</code>) e rode:</p>

```bash
docker-compose up
```

<p>A API estará disponível em <code>http://localhost:8080</code></p>
<p>A documentação completa da API pode ser acessada em <code>http://localhost:8080/swagger-ui/index.html</code></p>

<h3>Frontend</h3>

<p>Entre na pasta do frontend (<code>EducappFront</code>) e rode:</p>

```bash
npm install
```

<p>Depois inicie o Android Studio, emule um dispositivo Android e rode:</p>

```bash
npm run android
```

<h2>🇺🇸 English</h2>

<h1>Educap 📚</h1>

<p>A mobile application made to make communication and organization between students, teachers and the school easier.</p>

<hr>

<h2>About the project</h2>

<p>Educap was built with the idea that a school system would be much more practical as a mobile app than a website, because when a student wants to check tomorrow's tasks or a teacher wants to create an assignment, they will reach for their phone, not their computer.</p>

<p>The app is split into three access profiles: <b>Student</b>, <b>Teacher</b> and <b>Principal</b>, each with its own set of features. Some images of the app are available in the <code>assets</code> folder at the root of the project.</p>

<hr>

<h2>Features</h2>

<h3>🎒 Student</h3>
<ul>
  <li>Weekly class schedule displayed in a grid with a current time indicator</li>
  <li>Access to class tasks with filters and search</li>
  <li>Daily dashboard with next class and task summary</li>
  <li>School posts feed</li>
  <li>Class details with colleagues and teachers</li>
</ul>

<h3>👨‍🏫 Teacher</h3>
<ul>
  <li>Create, edit and remove tasks for their classes</li>
  <li>Weekly class schedule with classroom identification</li>
  <li>Access to class details</li>
  <li>Daily dashboard and school posts feed</li>
</ul>

<h3>🏫 Principal</h3>
<ul>
  <li>Access control: generate access keys and verify new accounts</li>
  <li>Classroom management</li>
  <li>Build the weekly class schedule grid</li>
  <li>Create posts for students and teachers</li>
</ul>

<hr>

<h2>Technologies</h2>

<b>Backend</b>
<ul>
  <li>Java with Spring Boot</li>
  <li>Spring Security + JWT</li>
  <li>PostgreSQL + PgAdmin4</li>
  <li>Docker</li>
  <li>Swagger OpenAPI</li>
</ul>

<b>Frontend</b>
<ul>
  <li>React Native</li>
</ul>

<hr>

<h2>How to run</h2>

<h3>Prerequisites</h3>
<ul>
  <li>Docker installed</li>
  <li>Android Studio installed</li>
</ul>

<h3>Backend</h3>

<p>Open the backend folder (<code>EducapBack/Educap</code>) and run:</p>

```bash
docker-compose up
```

<p>The API will be available at <code>http://localhost:8080</code></p>
<p>The full API documentation can be accessed at <code>http://localhost:8080/swagger-ui/index.html</code></p>

<h3>Frontend</h3>

<p>Open the frontend folder (<code>EducappFront</code>) and run:</p>

```bash
npm install
```

<p>Then open Android Studio, emulate an Android device and run:</p>

```bash
npm run android
```
